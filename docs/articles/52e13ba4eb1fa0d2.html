Overview

Red Hat build of Kueue 1.3 is now available. This Kubernetes-native job queuing controller manages how batch workloads share resources in a cluster. The 1.3 release adds support for specialized workload controllers and updates the API for future development.

Technical breakdown

After installing the operator, apply the following YAML to enable LeaderWorkerSet and JobSet :

apiVersion: kueue.openshift.io/v1 kind: Kueue metadata: labels: app.kubernetes.io/name: kueue-operator app.kubernetes.io/managed-by: kustomize name: cluster namespace: openshift-kueue-operator spec: managementState: Managed config: integrations: frameworks: - BatchJob - Pod - Deployment - StatefulSet - JobSet - LeaderWorkerSet Find installation instructions and release notes in the Red Hat documentation for Kueue.
