Overview

We’re excited to announce a new set of pre-written Sentinel policies for AWS designed to support ISO/IEC 27001 compliance. These policy sets aim to lower the barrier to adopting policy as code while helping organizations align their Terraform-managed AWS resources with globally recognized security standards. Co-created and co-owned by HashiCorp and AWS, these policies map to key ISO 27001 Annex A controls and are now available for use in the Terraform Registry.

Technical breakdown

These policies map to key ISO 27001 Annex A controls—covering areas such as access control, cryptography, logging and monitoring, and secure configuration management.

Getting started with pre-written Sentinel policies  Please refer to our  pre-written policy library documentation  to get started with pre-written Sentinel policies for ISO 27001, or check out the other pre-written Sentinel policy sets available in the Terraform registry:
